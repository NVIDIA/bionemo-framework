State-of-the-art models are continually integrated into the BioNeMo Framework. The BioNeMo Framework currently offers the following pre-trained models:

| **Model**                     | **Modality** | **Uses**                | **Training Location**                                                                    |
| ----------------------------- | ------------ | ----------------------- | ---------------------------------------------------------------------------------------- |
| [AMPLIFY](/models/amplify)       | Protein      | Representation Learning | [bionemo-recipes](https://github.com/NVIDIA/bionemo-framework/tree/main/bionemo-recipes) |
| [ESM-2](/models/ESM-2)     | Protein      | Representation Learning | [bionemo-recipes](https://github.com/NVIDIA/bionemo-framework/tree/main/bionemo-recipes) |
| [Untitled](/models/evo2)             | DNA          | Generative AI           | [bionemo-recipes](https://github.com/NVIDIA/bionemo-framework/tree/main/bionemo-recipes) |
| [Geneformer](/models/geneformer) | Single Cell  | Representation Learning | [bionemo-recipes](https://github.com/NVIDIA/bionemo-framework/tree/main/bionemo-recipes) |
